## Run Locally

Clone the project

```bash
  git clone https://github.com/hieungocnguyen/Ecommerce_FrontEnd
```

Go to the project directory

```bash
  cd Ecommerce_FrontEnd
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
  #or
  yarn dev
```
